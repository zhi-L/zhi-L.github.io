Python面向对象详解

面向对象的设计思想来自于自然界，将函数进行分类封装。 面向对象编程（Object Oriented Programming，OOP，面向对象程序设计） 面向对象的三大特点，封装、继承和多态。

一、封装

将实现某一特定功能的函数封装到一个类中。以便在其他地方可以更方便的调用。 首先看Python的面向对象中涉及的一些概念。

1、概念

  • 类：类是对现实世界中一些实物的封装。
  • 类对象：定义一个类之后便会产生一个类对象。
  • 实例对象：类实例化操作之后便会产生一个实例对象。
  • 类属性：类对象拥有的属性，被类对象的所有实例对象所共有，在内存中只存在一个副本。
  • 实例属性：为某个实例对象所特有。
  • 方法：与某个对象绑定的函数称作为方法，类中定义的函数一般与类对象或者实例对象绑定了，所以称之为方法。
  • 函数：类外定义的函数。

下面看一个具体的例子

class People:

    planet = "Earth"

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def printMessgae(self):
        print('country : {},'
              'name : {}, '
              'age : {}'.format(People.Planet, self.name, self.age))

if __name__ == "__main__":
    p = People('lz', 18)
    p.printMessgae()

上面定义了一个类People，则产生了一个类对象People。 通过类对象实例化了一个实例对象p。 country为类属性，name,age为实例属性。 printMessage()为方法。

2、属性

属性包括：

  • 类属性：属于类对象。在内存中只保存一份。
  • 实例属性：属于实例对象。在每个对象中都保存一份。

从上面的概念和例子中可看到他们在定义和使用中有所区别。本质的区别在于在内存中保存的位置不同。

3、方法

方法包括类方法，实例方法和静态方法。三种方法在内存中都归属于类，在内存中只保存一份。

（1）、类方法，实例方法以及静态方法

方法 装饰器 参数 说明
类方法 @classmethod 第一个参数必须是类对象，一般为cls 一般通过类对象调用，可以用来修改类属性。
实例方法 无 第一个参数必须为实例对象，一般为self 只能通过实例对象调用。
静态方法 @staticmethod 不需要定义参数 通过类对象调用。 
class People:

    planet = "Earth"
    __birth = 'niceMan'

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def printMessgae(self):
        print('country : {},'
              'name : {}, '
              'age : {}'.format(People.planet, self.name, self.age))

    @classmethod
    def getPlanet(cls):
        return cls.planet

    @classmethod
    def setPlanet(cls,planet):
        cls.planet = planet
        return cls.planet

    @staticmethod
    def getBirth():
        return People.__birth

上面例子中，printMessage()为普通方法，getPlanet()，setPlanet()为类方法。getBirth()为静态方法。

3、成员修饰符

成员包括属性和方法。

成员修饰符：

  • 公有成员：任何地方都可以访问。
  • 私有成员：只在类的内部可以访问。定义的方法是在命名时，在名字前面加两个下划线'__'
class People:

    planet = "Earth"
    __birth = 'niceMan'

    def __init__(self, name, age):
        self.name = name
        self.age = age

    def printMessgae(self):
        print('country : {},'
              'name : {}, '
              'age : {}'.format(People.planet, self.name, self.age))

    def __eat(self):
        pass

其中planet为公有属性，__birth为私有属性。 printMessage()位公有方法，__eat()为私有方法。 ps：如果想要强制访问私有字段，可以通过 【对象._类名__私有字段名 】访问（如：obj._C__foo），不建议强制访问私有成员。

二、继承

通过继承，子类可以继承并扩展父类的功能。 被继承的类称为父类、超类或者基类。继承的类称为子类。 Python继承支持多重继承，能够让一个子类拥有多个父类。多个父类之间用逗号隔开。

class UniversityMember:  

    def __init__(self,name,age):  
        self.name = name  
        self.age = age  

    def getName(self):  
        return self.name  

    def getAge(self):  
        return self.age  

class Student(UniversityMember):  

    def __init__(self,name,age,sno,mark):  
        UniversityMember.__init__(self,name,age)     #注意要显示调用父类构造方法，并传递参数self  
        self.sno = sno  
        self.mark = mark  

    def getSno(self):  
        return self.sno  

    def getMark(self):  
        return self.mark  



class Teacher(UniversityMember):  

    def __init__(self,name,age,tno,salary):  
        UniversityMember.__init__(self,name,age)  
        self.tno = tno  
        self.salary = salary  

    def getTno(self):  
        return self.tno  

    def getSalary(self):  
        return self.salary

子类实例化时，构造函数不会自动调用父类的构造函数。必须在子类中显示的调用。 可以通过以下两种方式调用父类的构造函数：

  • 父类.__init__()
  • super(子类， self).__init__()
class Base(object):
    def __init__(self):
        print "Base created"

class ChildA(Base):
    def __init__(self):
        Base.__init__(self)

class ChildB(Base):
    def __init__(self):
        super(ChildB, self).__init__()

三、多态

我理解Python的多态就是，子类可以直接继承使用父类的方法。同时也可以根据子类的需求，重写父类的方法。

  • 对扩展开放（Open for extension）：允许子类重写方法函数
  • 对修改封闭（Closed for modification）：不重写，直接继承父类方法函数
class Person(object):
     def __init__(self,name,sex):
         self.name = name
         self.sex = sex

     def print_title(self):
         if self.sex == "male":
             print("man")
         elif self.sex == "female":
            print("woman")

class Child(Person):                # Child 继承 Person
    def print_title(self):
        if self.sex == "male":
            print("boy")
        elif self.sex == "female":
            print("girl")

上面的例子中，父类根据性别的不同，分别输出male和female。 子类Child则重写了这一方法，根据性别的不同，分别输出boy和girl。
